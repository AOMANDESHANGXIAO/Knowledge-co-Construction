# 学生管理页面功能

- [ ] 学生可以创建团队，看见自己所处的团队
- [ ] 创建团队时可以填写的字段：团队名、团队颜色。
- [ ] 学生可以邀请其他同学参与自己的团队，生成一个团队码
- [ ] 学生可以通过团队码加入团队
- [ ] 点击某个按钮进行交互，输入团队码加入团队
- [ ] 学生可以看到自己团队正在参与的讨论
- [ ] 参与的讨论应当呈现卡片排列，可以搜索最近正在讨论的话题，并且可以按照时间排序呈现
- [ ] 学生可以看到自己团队讨论的情况
- [ ] 包含每个同学的：提出观点次数、反对观点次数、同意观点次数、整合观点次数以及整个团队的合计情况。
